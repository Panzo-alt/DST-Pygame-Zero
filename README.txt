======================================================

KNIGHT RUNNER

======================================================

I made a game called Knight Runner. It is a 2D runner that I created with Python using Pygame Zero. In this game the player has to avoid guys called Pokermad foes collect gold coins and stay away from bats that will reset the score.

------------------------------------------------------

LIBRARIES & DEPENDENCIES

------------------------------------------------------

To play this game you need to have some Python packages.

You already have these packages:

- this decides when enemies, bats and coins show up.

- math. This calculates the new score when you hit a bat.

You need to install these packages:

- pgzero. This helps with the game and graphics.

- pgzhelper. This makes it easy to animate the characters.

------------------------------------------------------

INSTALL AND RUN

------------------------------------------------------

To play the game follow these steps:

Step 1: Get Python

Go to python.org and download Python 3.

Step 2: Get the libraries

Open a terminal and type

pip install pgzero pgzhelper

Step 3: Play the game

Open the folder with the game files and type

python game.py in the terminal.

------------------------------------------------------

4. GAMEPLAY MECHANICS & ENEMIES

------------------------------------------------------

The goal of Knight Runner is to get a score and survive for a long time.. There are many things that can hurt you:

Pokermad foes are the bad guys on the ground. They are very dangerous.

* If you hit a Pokermad foe the game will be over.

* If you jump over a Pokermad foe and it goes off the screen you get one point.

Bats are the guys, in the air. They will not hurt you. They will stop you.

* If you hit a bat your score will be reduced to its root. For example if you have 100 points and you hit a bat your score will go down to 10. So try to avoid bats.

Gold coins are good.

* If you collect a gold coin you will get five points.
